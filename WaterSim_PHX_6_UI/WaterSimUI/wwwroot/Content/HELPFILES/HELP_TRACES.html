River Trace

River flows have long historical records. In WaterSim we use a 30-year segment from the historical record which we refer to as a "trace." If the simulation runs longer than 30 years, the trace segment chosen is repeated. The accompanying graph (immediately below) illustrates the annual river flows for the two riverine systems (and the median value for the trace; dashed line).

Each trace used will create different model results.
